# Inteligência Computacional Aplicada

Repositório dedicado aos códigos da disciplina de inteligência computacional aplicada

## Como usar

### 1. clonando o repositório

No diretório que deseja reproduzir estes códigos

```bash
git clone https://github.com/LuisFelipeCSouza/inteligencia-computacional-aplicada.git
```

### 2. Sincronizando ambiente virtual com [`uv`](https://github.com/LuisFelipeCSouza/inteligencia-computacional-aplicada.git)

```bash
cd ./inteligencia-computacional-aplicada
uv sync
```
